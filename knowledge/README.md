# ARTO Studio AI — Knowledge Base

Este directorio contiene la metodología, frameworks y conocimiento que hace a ARTO AI diferente de cualquier herramienta genérica. Es el cerebro del sistema.

## Estructura

```
knowledge/
├── methodology/          # Los 5 pilares de ARTO
│   ├── strategy.md       # Cómo ARTO hace strategy
│   ├── creativity.md     # Principios creativos
│   ├── narrative.md      # Cómo construimos narrativas
│   ├── production.md     # Estándares de producción
│   └── digital.md        # Frameworks digitales
├── industries/           # Patrones por industria (crece con cada cliente)
│   ├── fmcg.md
│   ├── tech.md
│   └── entertainment.md
├── quality/              # Rúbricas de auto-evaluación
│   ├── rubric.md         # Scoring 1-10 por pilar
│   ├── brand-voice.md    # Cómo suena ARTO
│   └── anti-patterns.md  # Qué NO hacer
├── trends/
│   └── current.md        # Tendencias actuales (Victor actualiza)
└── case-studies/         # Casos reales anonimizados
```

## Cómo Alimentar el Sistema

Victor y el equipo ARTO alimentan esta base de conocimiento con:
- Aprendizajes de proyectos reales
- Nuevos frameworks y metodologías
- Tendencias culturales y de mercado
- Feedback de clientes (anonimizado)
- Correcciones de calidad cuando el AI falla

**Cada actualización aquí mejora directamente el output para todos los clientes SaaS.**

## El Loop de Mejora

```
Proyecto real ARTO → Aprendizaje → Knowledge Base → ARTO AI mejora → Clientes SaaS
```
